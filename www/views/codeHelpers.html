Helpers

asObservable

A helper function that given a value ensure that the value is returned as an observable. If the value is already an observable it is simply returned if not a new observable is created wrapping that value. This is helpful when you want your code to use an observable but you want the developer to be able to pass a value or and observable value. 

    
self.value = ko.osObservable(possibleObservableValue);

asObservableArray

Same as asObservable but for observableArray. 

    
self.array = ko.osObservableArray(possibleObservableArray);
