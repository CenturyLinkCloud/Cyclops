Main Navigation api documentation

The <main-nav> is the primary navigation of a Cyclops-based application. It supports a simple two level system of navigation. For more information see the API documentation.

Responsive behavior is built in, so at narrow viewports (portrait tablet or mobile device) the links will collapse into an icon-based menu. At wide viewports, the icon and label will be adjacent instead of stacked.

In the iframe below, the main-nav is the menu bar on the left (Dashboard, Creat, Infrastructure, Network…)

Open Example Page 

    
<main-nav role="navigation" params="menus: menus, loading: loading, error: error, selectedItemId: selectedItemId"></main-nav>
    
    
$(function(){
  ko.applyBindings({
    menus: [
      {
        id: "home",
        name: 'Dashboard',
        iconId: '#icon-home',
        href: '#'
      },
      {...}
    ],
    self.loading = ko.observable(false);
    self.error = ko.observable(false);
    self.selectedItemId = ko.observable('infrastructure');
  });
});
